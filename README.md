🏥 Sistema de Gestão Hospitalar (Full Stack)
Este projeto consiste em uma plataforma robusta para o gerenciamento de fluxos hospitalares, integrando um Backend em Java (Spring Boot) com um Frontend dinâmico em React. O sistema foi desenvolvido para otimizar a triagem de pacientes e o atendimento médico em tempo real.

🚀 Tecnologias Utilizadas
Backend
Linguagem: Java.

Framework: Spring Boot.

Segurança/Arquitetura: APIs RESTful e persistência de dados.

Banco de Dados: PostgreSQL.

Frontend
Biblioteca: React.js.

Estilização: CSS3 e componentes modernos para Dashboards.

🛠️ Funcionalidades Principais
Painel de Recepção: Cadastro e triagem inicial de pacientes.

Painel Médico: Visualização de fila e prontuário para atendimento.

Integração de Dados: Comunicação fluida entre o cliente React e o servidor Spring Boot.

🏁 Como Executar o Projeto
Pré-requisitos
Java JDK 17 ou superior.

Node.js e npm instalados.

PostgreSQL configurado.

1. Configurando o Backend
Clone o repositório.

Importe o projeto na sua IDE (IntelliJ, Eclipse ou VS Code).

Configure as credenciais do seu banco de dados no arquivo application.properties.

Execute a classe principal HospitalManagementApplication.

2. Configurando o Frontend
Para iniciar a interface, siga a sequência de comandos abaixo no terminal (a partir da raiz do projeto):

Bash
# Navegar até a pasta do Front
cd '.\Front Sistema\'

# Entrar na pasta do projeto de recepção
cd recepcao-hospital

# Instalar as dependências (caso seja a primeira vez)
npm install

# Iniciar a aplicação
npm start
